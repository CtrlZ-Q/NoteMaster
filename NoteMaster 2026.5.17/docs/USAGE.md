# 使用教程

## 快速入门

### 1. 创建第一篇笔记

1. 启动 NoteMaster
2. 点击左侧边栏的「新建笔记」按钮
3. 在标题栏输入笔记标题
4. 在编辑区域开始编写 Markdown 内容

### 2. Markdown 基础语法

#### 标题

```markdown
# 一级标题
## 二级标题
### 三级标题
```

#### 文本样式

```markdown
**粗体文本**
*斜体文本*
~~删除线~~
`行内代码`
```

#### 列表

```markdown
- 无序列表项
- 第二项

1. 有序列表项
2. 第二项

- [ ] 待办事项
- [x] 已完成事项
```

#### 链接和图片

```markdown
[链接文本](https://example.com)
![图片描述](图片地址)
```

#### 代码块

````markdown
```javascript
function hello() {
  console.log('Hello, NoteMaster!');
}
```
````

#### 数学公式

```markdown
行内公式：$E = mc^2$

块级公式：
$$
\int_{-\infty}^{\infty} e^{-x^2} dx = \sqrt{\pi}
$$
```

#### 表格

```markdown
| 列1 | 列2 | 列3 |
|------|------|------|
| 内容 | 内容 | 内容 |
```

## 功能详解

### 文件夹管理

1. 在左侧边栏点击文件夹图标旁的「+」按钮
2. 输入文件夹名称并确认
3. 将笔记拖入文件夹，或在笔记详情中选择文件夹
4. 右键点击文件夹可重命名或删除

### 标签管理

1. 点击侧边栏底部的「标签管理」
2. 创建新标签并选择颜色
3. 在笔记编辑器中为笔记添加标签
4. 点击侧边栏中的标签可筛选相关笔记

### 搜索功能

- `Ctrl+F` 打开全局搜索
- 支持标题、内容、标签全文搜索
- 搜索结果实时高亮显示
- 基于 FlexSearch 的高性能搜索

### 编辑模式

| 模式 | 快捷键 | 说明 |
|------|--------|------|
| 编辑模式 | `Ctrl+1` | 只显示编辑器 |
| 预览模式 | `Ctrl+2` | 只显示预览 |
| 分屏模式 | `Ctrl+3` | 左右分屏显示 |

### AI 功能

#### 智能摘要

1. 打开一篇笔记
2. 点击工具栏的「AI摘要」按钮
3. 查看生成的摘要、关键词和知识点

#### 语法检查

1. 点击工具栏的「检查」按钮
2. 查看发现的语法问题和建议
3. 根据建议修改笔记内容

### 知识图谱

1. 点击「视图 > 知识图谱」或按 `Ctrl+G`
2. 查看笔记之间的关联关系
3. 点击节点可跳转到对应笔记
4. 使用鼠标滚轮缩放，拖拽移动视图

### 导出功能

1. 点击「文件 > 导出」
2. 选择导出格式（HTML/PDF/图片）
3. 配置导出选项
4. 点击「导出」按钮

## 数据备份

### 备份方法

1. 找到数据目录：
   - Windows: `%APPDATA%/NoteMaster/data/`
   - macOS: `~/Library/Application Support/NoteMaster/data/`
   - Linux: `~/.config/NoteMaster/data/`

2. 复制 `notemaster.db` 文件到安全位置

### 恢复方法

1. 关闭 NoteMaster
2. 将备份的 `notemaster.db` 复制回数据目录
3. 重新启动 NoteMaster

## 常见问题

### Q: 如何修改自动保存时间？

A: 在「设置」中调整「自动保存间隔」，默认为3秒。

### Q: 支持哪些编程语言的语法高亮？

A: 支持 180+ 种编程语言，包括但不限于：JavaScript、TypeScript、Python、Java、C++、Go、Rust、SQL、HTML、CSS 等。

### Q: 如何导入已有笔记？

A: 支持导入 `.md` 文件：
1. 点击「文件 > 导入」
2. 选择要导入的 Markdown 文件
3. 笔记将自动创建并保存

### Q: 知识图谱如何工作？

A: 知识图谱通过分析笔记的标签关系自动构建：
- 相同标签的笔记会显示连线
- 连线越粗表示共同标签越多
- 点击节点可跳转到对应笔记
